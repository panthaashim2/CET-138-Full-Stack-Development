HTML (HyperText Markup Language)

The Structure and Backbone of Web Pages

What is HTML?

HTML is the core language used to create and define the structure of web pages. It consists of semantic elements that organize content such as headings, paragraphs, lists, images, links, and multimedia.

Semantic HTML Elements

Using semantic tags improves accessibility, SEO, and makes code more readable:

<header>: Defines the introductory content or navigation links.

<nav>: Contains the navigation menu.

<main>: The main content area of the web page.

<article>: Defines a self-contained composition like a blog post.

<footer>: Contains footer information like copyright.

Example HTML Structure

<!DOCTYPE html>
<html>
  <head>
    <title>Sample HTML Page</title>
  </head>
  <body>
    <header>
      <h1>Welcome to My Website</h1>
    </header>
    <nav>
      <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
      </ul>
    </nav>
    <main>
      <p>This is a simple HTML page example.</p>
    </main>
  <footer>
    <p>Copyright 2025</p>
  </footer>
</body>
</html>

Forms, Tables, and Media

HTML supports input forms, tabular data, images, and multimedia content to create interactive and informative web pages.

Examples include <form> for input collection, <table> for data display, and <img>, <video>, <audio> for rich content.
