CSS (Cascading Style Sheets)

The Language that Styles Web Pages

What is CSS?

CSS is a stylesheet language used to describe the appearance, formatting, and layout of HTML elements on a webpage. It controls the colors, fonts, spacing, positioning, and responsive design to make webpages visually appealing.

Core Concepts

Selectors: Target HTML elements to apply styles.

Properties: Define what aspects to style (e.g., color, margin, font-size).

Box Model: CSS models every element as a box with content, padding, border, and margin.

Example CSS

body {
  font-family: Arial, sans-serif;
  background-color: #f0f4ff;
  color: #222;
}
h1 {
  color: #4b2a6a;
  text-align: center;
}
.container {
  width: 90%;
  max-width: 900px;
  margin: auto;
  padding: 20px;
  background-color: #fff;
  border-radius: 12px;
}

Advanced Styling

CSS supports responsive layouts via media queries, animations, flexbox, and grid systems. It also allows for pseudo-classes and pseudo-elements to add interactivity and decorative effects without extra HTML.
